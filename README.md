# Ground - Mario
> 시작은 똥피하기, 닷지, 블럭부수기를 리빌딩 해보는 것 이였지만.
> 어떻게 만들어 볼까 하다가 Wall World 라는 게임처럼 해보는건 어떤가 하는 의견이 나와서
> 회의를 하다보니 마리오의 땅굴파는 여행기가 되어버렸다.

## 사용 방법
  * 시작 씬 -> StartScene
  * 조작 방법
    * 이동 -> WAD, SHIFT
    * 조작 전환 -> SPACE
